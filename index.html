How to Install a Chrome Extension Manually

Step 1: Download The Zip

Click this here to download the zip.

Step 2: Unzip the File

Right-click the 'rbx api.zip' file and select 'Extract All...' or use your favorite unzipping tool.

Step 3: Install the Extension in Chrome

  1. Open the any browser.
  2. Right-click the exenstion icon
  3. Enable 'Developer mode' at the top right.
  4. Click the 'Load unpacked' button.
  5. Find the upload me folder.
  6. Select the Upload Me folder and upload it.
  7. Join any roblox game for the API to load.

Your extension should now be installed and visible in your extensions list.

For more information, visit Chrome Extension Development Guide.
